Get started / Introduction

Ozerus Design System

Un langage visuel éditorial et data-friendly pour l'extranet Ozerus : gestionnaire d'actifs, produits structurés, reportings partenaires. Palette vert sapin, marine, bleu brume, ochre sur sable chaud — typographie Familjen Grotesk + JetBrains Mono.

Principes

Hiérarchie typographique forte avec une serif pour les titres. Numéros en tabular-nums pour les données. Densité confortable, KPI avec sparklines, charts line simples, tables produits avec ISIN et valorisations. Sable chaud comme fond applicatif, crème pour les surfaces, marine pour les inversions ponctuelles.
Get started / Installation

Installation

Le design system est livré comme une collection de web components Stencil, consommables depuis n'importe quelle stack.

Vanilla HTML

<script type="module" src="node_modules/ozerus-ds/dist/ozerus-ds/ozerus-ds.esm.js"></script>

<oz-button variant="primary">Action</oz-button>

Angular / React / Vue

import { defineCustomElements } from 'ozerus-ds/loader';
defineCustomElements();

Tokens CSS

Les tokens sont injectés globalement par le loader et disponibles via var(--oz-*). Ils percent nativement le shadow DOM.

.my-custom {
  background: var(--oz-forest-50);
  color: var(--oz-forest-700);
  padding: var(--oz-s-4);
  border-radius: var(--oz-r-2);
  font-family: var(--oz-font-sans);
}
